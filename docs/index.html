Website preview

What the web version can show

This GitHub Pages site is a public preview and documentation surface. The full trainer still needs the local desktop app because Chess.com imports, Stockfish analysis, cache storage, and Lichess token use require a backend running on your machine.

Choose any legal move from your side. Bookup follows your repertoire branch, classifies the move, then replies from the live position.

Run Bookup

Use the desktop app for the full trainer.

The root-level executable is the main launch path for Windows builds. Developers can also run the Flask app from source with Python and a local Stockfish binary. 

Bookup.exe

py -3.13 -m pip install -r requirements.txt
py -3.13 main.py
